# Randicon

Randicon generates gravatar-like, slack-like icons based on the string hashcode.

## Usage

````html
  <script src="randicon.min.js"></script>


  <svg id="icon1" width="64" height="64"/>

  <script>
    randicon("icon1", "johnny-bravo@acme.com");
  </script>
  ````

## Development

````
npm install
bower install
gulp
````
